The pass, in order

Steady state first — it's cheap, it has no time-resolution artifacts to misread, and it tells you where to point everything that comes after. Then time-resolved emission, then pump-probe. Each part leans on the previous one.

Part 1 — Steady state Read the .asc, calibrate the axes (real space or k-space), reflectance contrast & backgrounds, lineshapes (Lorentzian / Gaussian / Fano + a TMM sanity check), PL and its spatial profile, then sweeps: power / temperature / gate.
Part 2 — TRPL & PL imaging TCSPC histograms and the IRF, analytic IRF-convolved fits, spatial profiles from a point scan or a detector array, MSD analysis — and everything that broadens a profile without being transport.
Part 3 — Transient reflectance What dR/R actually measures (not always population!), wavelength calibration with filter edges, chirp correction, and the full interactive map analysis: spatial fits, dynamics, MSD → D.

Rule zero throughout: every pipeline here runs on synthetic data with known ground truth before it ever touches a measurement. If it can't recover what was put in, it doesn't get used.

1.1Reading the .asc and calibrating the axes

The raw file is a matrix: first column wavelength, the rest are pixels along the slit. The spectral axis comes calibrated; the spatial axis is where the thinking happens. Real space: µm/pixel = camera pitch / magnification (f_tube / f_objective) — cross-checked by imaging a diffraction-limited laser spot of known size (~0.51 λ/NA FWHM; the measured spot can only be bigger, so this bounds the calibration). k-space: if the back focal plane is imaged, the edge of the illuminated disk sits exactly at the objective NA, and position is linear in sin θ — so the angle axis stretches toward the edge.

def read_asc(text):
    rows = []
    for line in text.splitlines():
        line = line.strip()
        if not line or line.startswith("#"):
            continue
        rows.append([float(p) for p in line.replace(",", "\t").split()])
    arr = np.asarray(rows)
    return arr[:, 0], arr[:, 1:]        # wavelength_nm, (n_wl, n_px) image

def real_space_um_per_px(camera_pitch_um=16.0, f_objective_mm=2.0, f_tube_mm=200.0):
    return camera_pitch_um / (f_tube_mm / f_objective_mm)

def k_space_angle_axis(n_px, edge_lo_px, edge_hi_px, NA=0.8):
    """BFP disk edge = NA; position is linear in sin(theta), not theta."""
    center = 0.5 * (edge_lo_px + edge_hi_px)
    radius = 0.5 * (edge_hi_px - edge_lo_px)
    sin_theta = np.clip(NA * (np.arange(n_px) - center) / radius, -0.999, 0.999)
    return np.degrees(np.arcsin(sin_theta))

1.2Do you need k-space at all?

One question decides it: does the feature disperse inside the light cone your objective collects? Drag the slider from flat to strongly dispersing. A flat feature (most bare emitters) makes k-space imaging a waste of a measurement — real space tells you more. A bending feature (cavity modes, polaritons, anything with structure vs angle) makes k-space mandatory, because a single-angle spectrum samples an arbitrary point of a curve you can't reconstruct. (The full k-space toolkit — dispersion fitting and friends — is deferred to a future part.)

1.3aReflectance contrast and the background problem

RC = (R_sample − R_reference) / R_reference kills the lamp spectrum and the system response — if you have a reference. With an off-sample / bare-substrate region, you measure it and life is good. Without one (a planar cavity is everywhere, and its "background" is thickness-dependent), the reference must be estimated. Toggle the checkbox and watch how much the recovered feature depends on the interpolation choice.

Open question: the best background estimate without a reference region. My current approach: mask the feature, interpolate through the feature-free wavelengths with a low-order polynomial, and report the spread over orders as a systematic. Better ideas welcome.

def estimate_background(E, R, feature_window=(1.67, 1.73), order=1):
    """No reference region: mask the resonance, fit a polynomial through
    the rest. Try a couple of orders; the spread IS the systematic."""
    m = (E < feature_window[0]) | (E > feature_window[1])
    coeffs = np.polyfit(E[m], R[m], order)
    return np.polyval(coeffs, E)

B = estimate_background(E, R_sample, order=2)
rc_estimated = (R_sample - B) / B

1.3bLineshapes: Lorentzian, Gaussian, Fano

Lorentzian = one damped resonance (heavy 1/ΔE² tails). Gaussian = inhomogeneous broadening (tails die fast). Fano = a discrete resonance interfering with a continuum — the generic case for a thin resonant layer on a dielectric stack. One formula covers the family: F ∝ (q+ε)²/(1+ε²) with ε = 2(E−E₀)/Γ. Drag q: at q=0 it's a pure dip, at large q a Lorentzian peak, in between the classic asymmetric shape whose extremum is displaced from E₀ by ~Γ/2q — reading the position off the plot fails exactly when the shape is asymmetric. Fit-recipe for p0/bounds: center from the (smoothed) extremum, bounded to the window; width from counting samples above half-max, bounded below by instrument resolution; amplitude from extremum-minus-baseline. A parameter pinned at a bound = wrong model or window.

Python — the three lineshapes + the p0/bounds recipe 
def lorentzian(E, A, E0, G, c):
    return A * (G / 2) ** 2 / ((E - E0) ** 2 + (G / 2) ** 2) + c

def gaussian(E, A, E0, G, c):                        # parametrized by FWHM
    return A * np.exp(-0.5 * ((E - E0) / (G / 2.355)) ** 2) + c

def fano(E, A, E0, G, q, c):
    eps = 2.0 * (E - E0) / G
    return A * (q + eps) ** 2 / (1.0 + eps ** 2) + c

def initial_guess(E, y):
    base = np.median(np.concatenate([y[:20], y[-20:]]))
    i_ext = int(np.argmax(np.abs(y - base)))         # center: the extremum
    A0 = y[i_ext] - base                             # amplitude vs baseline
    above = E[np.abs(y - base) > 0.5 * abs(A0)]      # width: half-max span
    G0 = max(above.max() - above.min(), (E[1] - E[0]) * 3)
    return base, E[i_ext], A0, G0

1.3cWhy RC looks Fano: a 30-line transfer-matrix check

Put a perfectly symmetric Lorentz oscillator in a thin layer (the textbook case: an hBN-encapsulated monolayer on SiO₂/Si) and compute the RC with a transfer matrix: the lineshape comes out asymmetric, and its sign and shape swing with the oxide thickness — pure interference between the resonant reflection and the rest of the stack. So before reading physics into an asymmetric or inverted RC lineshape, check the stack. Live TMM below: drag the oscillator strength and the spacer thickness.

Python — the TMM core 
def reflectance_stack(E_eV, layers, n_in=1.0, n_sub=3.88 + 0.02j):
    """layers = [(refractive_index, thickness_nm), ...], normal incidence."""
    lam_nm = 1239.841984 / np.asarray(E_eV)
    m11 = np.ones_like(lam_nm, dtype=complex); m12 = np.zeros_like(m11)
    m21 = np.zeros_like(m11);                  m22 = np.ones_like(m11)
    for n, t_nm in layers:
        n = np.asarray(n, dtype=complex) * np.ones_like(m11)
        d = 2 * np.pi * n * t_nm / lam_nm
        c, s = np.cos(d), np.sin(d)
        m11, m12, m21, m22 = (m11*c + m12*1j*n*s, m11*1j*s/n + m12*c,
                              m21*c + m22*1j*n*s, m21*1j*s/n + m22*c)
    num = n_in * (m11 + m12 * n_sub) - (m21 + m22 * n_sub)
    den = n_in * (m11 + m12 * n_sub) + (m21 + m22 * n_sub)
    return np.abs(num / den) ** 2

def lorentz_index(E, eps_b=6.0, f=0.8, E0=1.70, gamma=0.010):
    return np.sqrt(eps_b + f / (E0**2 - E**2 - 1j * gamma * E))

1.4Steady-state PL: the counts and the profile

Two reminders before interpreting PL. (i) PL is time-integrated. With a pulsed laser, counts ∝ rep-rate × photons-per-pulse: the same average power at half the rep rate doubles the per-pulse density, so anything nonlinear changes "the same measurement". And integrated intensity ∝ n₀·τ — a sample can dim purely because a nonradiative channel shortened τ. (ii) The profile is not the laser. Whatever is excited moves before it emits: for diffusion during an exponential lifetime the variances add, σ²_PL = σ²_laser + 2Dτ, giving the diffusion length from two second moments (a Gaussian fit is core-weighted and biased low once the time-integrated profile grows heavy wings — the readout below uses moments). Peak fitting itself: same recipe as reflectance, §1.3b.

def steady_state_profile(x, sigma_L, D, tau, n_t=400):
    """I(x) ~ integral dt e^{-t/tau} N(x; sigma_L^2 + 2 D t)"""
    t = np.linspace(0, 8 * tau, n_t)
    var = sigma_L ** 2 + 2 * D * t
    w = np.exp(-t / tau)
    return np.trapz(w / np.sqrt(var) *
                    np.exp(-0.5 * x[:, None] ** 2 / var), t, axis=1)

def moment_sigma(x, y):
    """Variance obeys the sum rule exactly; a Gaussian FIT does not (it is
    core-weighted, biased low on heavy-winged profiles). Subtract the
    baseline; do NOT clip negative noise (clipping rectifies the wings)."""
    yy = y - np.median(y[np.abs(x) > 0.8 * x.max()])
    mu = np.sum(x * yy) / np.sum(yy)
    return np.sqrt(np.sum((x - mu) ** 2 * yy) / np.sum(yy))

L_D = np.sqrt((moment_sigma(x, pl) ** 2 - moment_sigma(x, laser) ** 2) / 2)

1.5Sweeps: tracking peaks vs power / temperature / gate

Every sample gets swept against something, and the data becomes a stack of spectra vs a control knob. To track a species through the sweep I fit sequentially: step k's result seeds step k+1's initial guess, with bounds tightened to a window around it. Identity is preserved by continuity — a physical assumption, so it fails (deliberately) at a genuine abrupt transition; watch the residuals for that. Independent per-step fits with a generic p0 swap peak identities as soon as two peaks overlap. Drag the knob through the crossing region below.

def fit_sequential(E, spectra, mu_window=0.006, amp_factor=2.0):
    """Previous popt becomes p0; bounds tighten around it."""
    out, prev = [], None
    for spec in spectra:
        if prev is None:
            p0, lo, hi = generic_first_step()
        else:
            p0 = list(prev)
            lo = [max(prev[0] / amp_factor, 0), prev[1] - mu_window, 0.003,
                  max(prev[3] / amp_factor, 0), prev[4] - mu_window, 0.003]
            hi = [max(prev[0] * amp_factor, 50), prev[1] + mu_window, 0.03,
                  max(prev[3] * amp_factor, 50), prev[4] + mu_window, 0.03]
            p0 = [min(max(p, l), h) for p, l, h in zip(p0, lo, hi)]

2.1–2.2The TCSPC histogram and IRF-convolved fitting

A PicoHarp / SPAD file is fundamentally two numbers (time bin width, number of channels) plus counts. The histogram is the decay convolved with the instrument response function — laser pulse + detector jitter + electronics, typically tens to hundreds of ps. Never deconvolve (it amplifies noise); fit the analytic convolution of exponentials with a Gaussian IRF (the erfc form below), weighted by √counts, with the dark floor taken from pre-pulse bins. Cross-check the fitted lifetimes with the model-free 1/e time.

from scipy.special import erfc

def _conv_term(t, A, tau, t0, w):
    """exp(-t/tau)*H(t) convolved with a Gaussian IRF of sigma w."""
    arg_exp  = (w * w - 2.0 * tau * (t - t0)) / (2.0 * tau * tau)
    arg_erfc = (w * w - tau * (t - t0)) / (np.sqrt(2.0) * w * tau)
    out = np.zeros_like(t, dtype=float)
    safe = arg_exp < 700.0
    out[safe] = 0.5 * A * np.exp(arg_exp[safe]) * erfc(arg_erfc[safe])
    return out

def biexp_irf(t, y0, t0, A1, tau1, A2, tau2, w):
    return y0 + _conv_term(t, A1, tau1, t0, w) + _conv_term(t, A2, tau2, t0, w)

sigma = np.sqrt(np.clip(counts, 1, None))            # Poisson weights
popt, pcov = curve_fit(biexp_irf, t, counts, p0=p0, bounds=(lo, hi), sigma=sigma)

2.3Spatial profiles two ways: point scan vs detector array

Route A: single-pixel detector + two stages — park, record a full decay, move, repeat over a grid. Full time resolution everywhere, but slow (N² acquisitions) and drift-sensitive. Route B: a detector array (ours has 23 pixels) records everywhere at once; it doesn't image the spot, it samples it, and a 2D Gaussian fit per time bin turns the samples into σ_x, σ_y — carried around as σ_eq = √((σ_x²+σ_y²)/2). Both routes end at the same deliverable: σ(t), the input for the MSD analysis below. The script simulates both and shows they agree (the sparse array is noisier at late times, as it should be).

Python — the 2D Gaussian fit both routes share 
def gauss2d(coords, A, x0, y0, sx, sy, offset):
    x, y = coords
    return (A * np.exp(-(((x - x0)**2) / (2 * sx**2)
                       + ((y - y0)**2) / (2 * sy**2))) + offset)

def fit_gauss2d(x, y, z):
    p0 = [z.max() - z.min(),
          np.average(x, weights=z - z.min() + 1e-12),
          np.average(y, weights=z - z.min() + 1e-12), 1.0, 1.0, z.min()]
    popt, _ = curve_fit(gauss2d, (x, y), z, p0=p0, bounds=(lo, hi))
    sigma_eq = np.sqrt((popt[3] ** 2 + popt[4] ** 2) / 2)
    return popt, sigma_eq

2.4MSD analysis: broadening is not always transport

Fit σ(t), plot MSD = σ(t)² − σ(0)², read the physics off the shape — carefully. Four mechanisms broaden a profile and only two involve motion: diffusion (MSD = 2Dt, linear), ballistic (v²t², superlinear), annihilation (bimolecular decay eats the dense center first — the profile flattens and σ grows with zero transport), and emission saturation (detected signal clips at the center early on — apparent σ starts high and shrinks: "negative diffusion"). Pick a mechanism and scrub time. My acceptance test before claiming a D: MSD linear over a decade of time, AND the extracted D independent of pump power. A power-dependent "D" is the classic annihilation fingerprint.

2.5Two species, one cascade: spectrally-resolved imaging

With two emission bands, a band-pass in the imaging path gives one spatial profile per species. A toy model worth internalizing: the pump creates only the high-energy species H; H relaxes into low-energy L; L has finite capacity, so the transfer saturates. Prediction: H stays laser-shaped, L comes out broader and flat-topped at high pump — the center hits capacity and clips, the wings don't — and L's rise at the center is slower than in the wings. Crank the pump. A flat-top or donut in one spectral window with a normal Gaussian in the other is a saturation fingerprint; the low-pump check (left end of the slider) is what separates it from transport.

pump level
Python — the cascade rate equations 
# dn_H/dt = -n_H/tau_H - k n_H (1 - n_L/N_sat)
# dn_L/dt = -n_L/tau_L + k n_H (1 - n_L/N_sat)
for i, t in enumerate(times):
    transfer = k * nH * np.clip(1.0 - nL / N_sat, 0.0, None)
    nH = nH + dt * (-nH / tau_H - transfer)
    nL = nL + dt * (-nL / tau_L + transfer)
    H_integrated += nH / tau_H * dt      # what the filtered camera sees

pump-probe maps — and why the signal is not automatically population

3.1What makes a dR/R signal

dR/R₀ = (R_on − R_off)/R_off. The pump can do three independent things to the resonance, each with its own fingerprint: an amplitude change (bleach or induced absorption) gives one lobe shaped like the resonance itself — the population-like term; a spectral shift gives a dispersive two-lobe first-derivative shape; broadening gives a three-lobe second-derivative shape. Shift and broadening produce large signals with essentially zero population change. Mix them below — then imagine sampling this at a single wavelength and calling it "the dynamics". That's why maps beat single-wavelength kinetics: a trace on a shifting resonance can flip sign while the population decays monotonically.

def reflectance_dip(E, A=0.15, E0=1.70, G=0.012, base=0.35, slope=-0.15):
    return base + slope * (E - 1.70) - A * (G/2)**2 / ((E - E0)**2 + (G/2)**2)

def pumped(E, bleach=0.0, shift=0.0, broaden=0.0):
    return reflectance_dip(E, A=0.15 * (1 - bleach),
                           E0=1.70 + shift, G=0.012 * (1 + broaden))

dr_over_r = (pumped(E, **modification) - R0) / R0

3.2Calibration and chirp correction

Raw camera files (ours come as paired "para"/"sum" text files: delays in the para header, frames concatenated in the sum) reshape to a (delay, y, wavelength) stack. Two preprocessing steps before any physics. Wavelength: drop long/shortpass filters into the white-light probe — it truncates exactly at the filter edge, the 50% crossing pixel maps to the edge wavelength, two filters give the two-point linear calibration. Chirp: group-velocity dispersion makes t₀ wavelength-dependent (the bent onset below); fit a polynomial t₀(λ) to the onset / coherent artifact and re-interpolate each wavelength column onto a common time axis. Crank the chirp and toggle the correction.

chirp strength
Python — filter-edge calibration + chirp correction 
def edge_crossing_pixel(profile):
    """White light truncates at the filter edge; find the 50% crossing."""
    half = 0.5 * np.percentile(profile, 90)
    i = int(np.argmax(profile > half))
    y0, y1 = profile[i - 1], profile[i]
    return (i - 1) + (half - y0) / (y1 - y0)
# two filters -> two (pixel, lambda) anchors -> linear axis

def chirp_correct(t, wl, data, coeffs, dt_uniform=0.1):
    t_uni = np.arange(t[0], t[-1], dt_uniform)
    data_uni = np.stack([np.interp(t_uni, t, data[:, j])
                         for j in range(wl.size)], axis=1)
    t0 = np.polyval(coeffs, wl); t0 -= t0.mean()
    idx = (t_uni[:, None] + t0[None, :] - t_uni[0]) / dt_uniform
    coords = np.stack([idx, np.broadcast_to(np.arange(wl.size), idx.shape)])
    corrected = map_coordinates(data_uni, coords, order=1, mode="nearest")
    return crop_to_common_window(t_uni, corrected, t0, dt_uniform)

3.3The full map analysis, live

Everything converges here: a (delay, y, λ) stack with a weak signal that decays and diffuses. Click the map to move the crosshair, scrub the delay. The spatial cut gets a live Gaussian fit → σ(t); the dynamics panel keeps the pre-t₀ points (they establish the baseline and expose pump scatter) and fits a plain exponential after the rise — no IRF convolution needed here, because pump-probe time resolution is the optical cross-correlation of two ~100 fs pulses, orders of magnitude below the dynamics, unlike a TCSPC detector IRF which competes with the lifetime. The MSD panel turns σ(t) into a D — subject to every caveat from §2.4. One more trap, visible right here: the fitted τ at the spot center comes out shorter than the population lifetime (12 ps in this synthetic stack), because diffusion carries signal out of the point you're watching. Local decay ≠ population decay whenever anything moves — transport lives in the MSD panel, not in a single-point τ.
